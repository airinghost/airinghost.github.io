站点初创：

域名里的 Zhang 是姓氏拼音，Jet 是以前读培训班时从课本词汇表里挑的最短的英文名。最后选的 .COM 是因为它最常用。

始于20190109《吃火锅记》。

眼花缭乱：

为了把网站搭起来，用过好多服务：anyNode VPS / Vultr VPS / 阿里云ECS / 腾讯轻量云 / 华为云Flexus / GitHub Pages / CODING Pages / Gitee Pages / Netlify / Vercel / Cloudflare / 腾讯云CDN / 多吉云。

东奔西走：

WordPress 阶段：因为 VPS 面板很多都提供 WP 的一键安装功能，所以我与许多网友的选择一样。主题用的是朴素的 Adams。

Jekyll 阶段：因为不想再续费 VPS，所以喜欢上免费的 Pages 服务，而 GitHub 和 Gitee Pages 默认就支持 Jekyll。当时有两款主题挺喜欢，分别叫 HardCandy 和 TMaize，可是配置文件看不懂，只能乱搞一通，遂放弃，转向 Hexo。

Hexo 阶段：主题仍与很多人一样选择了 NexT。但后来我觉着它顶部的导航栏每次都要点击下拉，这个操作在移动端上实在繁琐；又恰巧在 MoeCraft 的用户中心网站页脚看见了 Icarus 主题的链接，点进去瞧瞧发现很好看！于是果断换之，胡乱删改样式，用了两年。缺点就是生成有点慢。

Hugo 阶段：确实快！而且只需一个可执行文件，没有厚重的 node_modules。刚开始准备用时，看中了一款叫 Stack 的主题，但在移动端上它也不能直接显示导航栏，也得点击下拉，而且模板看不懂、改不动，所以继续用 Hexo。之后想了想，反正 Icarus 也手动改过，与其老是删改缝补，不如根据改过的样式自己撸一个 Hugo 主题吧。说干就干，边敲键盘边看文档边调试，鼓捣出一个简陋的 Icarus-Lite 也放在了 GitHub 上。

最后（目前）是手打 HTML，修改灵感部分来自 flomo。网页字体使用了“文泉驿点阵宋体12px”，由ZeoSeven 提供的托管。

怦然心动：

LOGO 原本用的是我在手机备忘录随手描了几笔的笑脸，觉得很有趣就留了下来。但源文件后来弄丢了，截图又很糊，所以现在直接简单地用了一个 Emoji 💖

如果要外链的话，可以引用网站根目录下的 favicon.svg

时刻关注：

之前根据 @Xing 同学的建议，通过 Hugo 内置的 RSS 模板添加过订阅功能，但之后没有再用 Hugo，所以订阅暂时消失了好久～

最后在 @Melatonin 同学的关心催促下， RSS 订阅又上线了！访问网站根目录下的 rss.xml 即可。
